Abstract:

Neste trabalho era pretendido criar mais features para a aplicação realizada no TPC3 que permitisse ao utilizador ver os resultados obtidos das queries em tabelas.

Na tabela apresentada com os resultados é possivel ainda ver e executar novas queries nos elementos linkados, á semelhança do que acontece no GraphDB.

O utilizador pode ainda exportar os resultados de qualquer query.

Para a interface disponibilizada ao utilizador foi criado um Servidor em ReactJS. Neste caso, é este quem gere toda a aplicação do lado do utilizador e o servidor NodeJS funciona como uma "ponte" que estabelece comunicação entre o GraphDB e o servidor de interface, mantendo assim toda a validação das interações entre os dois sistemas.

Para instalar basta exectuar o comando "npm i" em ambas diretorias e de seguida "npm start" em ambas. De preferência começar sempre por executar o Servidor API de dados.

Deliverables:

  • Servidor API de Dados (NodeJS)
  • Servidor Interface (ReactJS)
