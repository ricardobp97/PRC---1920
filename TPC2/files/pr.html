Abstract:

Neste trabalho foi criada uma ontologia para o ficheiro de arquivo musical. Para isso foi utilizado o Protégé que permite criar ontologias a partir de uma interface gráfica.

Assim que obtido o formato da ontologia através do ficheiro TTL, foi criado um gerador em XSLT para converter todos os dados do ficheiro do arquivo musical de XML para TTL.

De seguida foi importado o ficheiro com a ontologia, preenchida com todos os dados, diretamente no GraphDB. As queries SPARQL enunciadas para este trabalho foram efetuadas com sucesso, estando disponiveis nos ficheiros abaixo.

Deliverables:

  • Arquivo Musical (XML)
  • Gerador para TTL (XSLT)
  • Ontologia (TTL)
  • Queries SPARQL
