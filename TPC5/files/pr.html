Abstract:

Neste trabalho era pretendido criar um Dataset de Nadadores, a partir da informação disponivel na DBPedia.

Foi então retirada toda a informação através da query disponibilizada abaixo. Os dados foram retirados em formato JSON.

Contudo, foi preciso ainda tratar estes dados pois continham nadadores repetidos e alguns elementos em formatos não desejados. Para isso foi criada uma script em JavaScript para normalizar o dataset.

Abaixo é disponibilizado o Dataset final, mas também o primeiro retirado diretamente da DBPedia e a script para o seu tratamento e transformação final.

Deliverables:

  • JSON Obtido
