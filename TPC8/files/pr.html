Abstract:

Neste trabalho prático havia duas grandes tarefas: (1) A criação de uma API em NodeJS para o dataset de Cinema (2) A criação de uma WebApp para efetuar queries Sparql Construct e visuapzar os resultados ou inseri-los diretamente na base de dados GraphDB.

A primeira tarefa foi reapzada com sucesso com a criação de uma API em NodeJS para aceder aos dados do dataset cinema2020 carregado no GraphDB que disponibipza as seguintes rotas:

/filmes e /filmes/:id

/atores e /atores/:id

/personagens e /personagens/:id

/reapzadores e /reapzadores/:id

/produtores e /produtores/:id

/escritores e /escritores/:id

/paises e /paises/:id

/pnguas e /pnguas/:id

/generos e /generos/:id

Para a segunda tarefa foram criados dois servidores:

Servidor API: responsável por efetuar os pedidos ao GraphDB

Servidor UI: responsável por disponibipzar uma interface ao utipzador para escrever as queries e visuapzar os seus resultados

Para utipzar esta WebApp basta pgar o GraphDB, o Servidor API e por último o Servidor UI.
